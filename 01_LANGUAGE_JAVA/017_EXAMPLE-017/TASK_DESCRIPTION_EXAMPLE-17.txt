==============================================
DESCRIBE ENGLISH
==============================================

a program with a graphical interface

Create a program that animates a circle and a square. 
The circle should get bigger and bigger from the inside out and 
the square should get smaller and smaller from the inside out. 
The animations should be repeated 10 times and automatically 
run backwards. All other settings for the animation, 
such as size or speed, can be set by you.
