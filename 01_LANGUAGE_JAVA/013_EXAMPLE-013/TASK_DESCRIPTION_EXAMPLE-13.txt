DESCRIBE ENGLISH 
==============================================

1. create an editor that allows you to save a list of 
words for the Hangman game to a file. In this file you will 
save not only the words themselves, but also the number of words saved. 
It is enough to append a new word to the end of the file to save it. 
Subsequent changes to saved words are not necessary. However, 
the editor should be able to display the existing words. 
How you implement the editor in detail and which library
you use for the interface is up to you. However, please 
document your solution in detail.
2. Modify the Hangman game from this tutorial so that the 
word is no longer randomly picked from an array, 
but from the file you created in Task 1. However, 
the word should still be determined randomly. 
How you implement the access is up to you. However, 
please document this solution in detail.
