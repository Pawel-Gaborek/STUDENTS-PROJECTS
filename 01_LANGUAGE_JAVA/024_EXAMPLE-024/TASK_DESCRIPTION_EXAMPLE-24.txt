DESCRIBE ENGLISH 
==============================================

a program with a graphical interface

1. extend the application from this study booklet, in which you can change the size of a 
text using the two buttons, so that the current size is also displayed in a label.
Arrange the components in a grid layout with two columns. It is up to you which 
component appears where.
Please note that you cannot display the value of an int type directly in a label. 
You must first convert the value to the string type - for example, 
with Integer.toString(<intValue>).
