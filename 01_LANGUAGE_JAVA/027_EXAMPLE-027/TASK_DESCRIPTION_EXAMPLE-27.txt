DESCRIBE ENGLISH 
==============================================

a program with a graphical interface

1. formulate catch statements to handle the following exceptions
a) An argument is passed in an invalid format,
b) an attempt to access a non-existent class,
c) a failed conversion of a string to a numeric value,
d) access to an invalid index in an array,
e) for any real exceptions that might occur.
You only need to specify the line with the catch statement.
