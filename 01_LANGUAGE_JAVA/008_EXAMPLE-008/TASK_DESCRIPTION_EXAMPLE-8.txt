The circle should get bigger and bigger from the inside out and the square should get smaller and smaller from the inside out. The animations should be repeated 10 times and automatically run backwards. All other settings for the animation, such as size or speed, can be set by you.==============================================DESCRIBE GERMAN // BESCHREIBUNG AUF DEUTSCH==============================================Erstellen Sie ein Programm, das einen Kreis und ein Quadrat animiert.
