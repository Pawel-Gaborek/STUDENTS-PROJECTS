a program with a graphical interface

Convert the Swing Editor to an applet. For the toolbar, 
make sure that the images for the icons are loaded correctly. 
Create an HTML file that launches the applet for testing purposes. 
Send this HTML file along with your solution. You can use the file 
created by the applet viewer as a template. 
It is sufficient for testing if the applet runs in the Applet Viewer. 
You do not need to test it in the browser.
